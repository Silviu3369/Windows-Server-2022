CMD util - pack practic

Concentrat pe diagnostic rapid și task-uri de urgență.

Focused on fast diagnostics and urgent operations.

L1. Network quick checks

Teorie: cu 5 comenzi poți identifica majoritatea problemelor de conectivitate.

Theory: with 5 commands you can identify most connectivity problems.

ipconfig /all
ping 8.8.8.8
tracert 8.8.8.8
nslookup academy.local
netstat -ano

L2. System health

Teorie: baseline-ul de sănătate se colectează rapid cu systeminfo + SFC + DISM.

Theory: health baseline is quickly collected via systeminfo + SFC + DISM.

systeminfo
sfc /scannow
DISM /Online /Cleanup-Image /RestoreHealth

L3. Files și procese

Teorie: în incidente, căutarea rapidă în loguri și identificarea proceselor e vitală.

Theory: in incidents, fast log search and process identification are critical.

dir /s *.log
findstr /i error *.log
tasklist
taskkill /PID 1234 /F

L4. Service control

Teorie: comanda SC permite controlul direct al serviciilor când GUI nu e disponibil.

Theory: SC command allows direct service control when GUI is unavailable.

sc query
sc query w32time
sc start w32time
sc stop w32time
