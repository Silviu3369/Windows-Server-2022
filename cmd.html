CMD Essentials

Toolkit minimal pentru incidente urgente cand ai nevoie de raspuns rapid.

1. Networking

ipconfig /all
ping 8.8.8.8
tracert 8.8.8.8
nslookup academy.local

2. System

systeminfo
sfc /scannow
DISM /Online /Cleanup-Image /RestoreHealth

3. Files & Process

dir /s *.log
findstr /i error *.log
tasklist
taskkill /PID 1234 /F

4. Services

sc query
sc query w32time
sc start w32time
sc stop w32time
