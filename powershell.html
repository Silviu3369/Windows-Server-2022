PowerShell - invatare structurata

Capitolele sunt gandite pentru administratori IT, cu aplicatie directa in server/client/cloud.

1. Fundamentals

Inveti modelul Verb-Noun, obiecte si help system.

Get-Command
Get-Help Get-Service -Full

2. Pipeline

Get-Service | Where-Object Status -eq "Running" | Select Name,Status
Get-Process | Sort CPU -Descending | Select -First 10

3. Scripting

param([string]$Name="World")
Write-Host "Hello $Name"

4. Modules

Get-Module -ListAvailable
Import-Module ActiveDirectory

5. Error Handling

try { Get-Item C:\missing.txt -ErrorAction Stop } catch { $_ }

6. Remoting

Enable-PSRemoting -Force
Invoke-Command -ComputerName SRV-DC01 -ScriptBlock { Get-Service }

7. Automation

Register-ScheduledTask -TaskName "DailyHealth" -Action (New-ScheduledTaskAction -Execute "powershell.exe" -Argument "-File C:\Scripts\health.ps1") -Trigger (New-ScheduledTaskTrigger -Daily -At 08:00) -User "SYSTEM" -RunLevel Highest

8. Reporting

Get-ADUser -Filter * | Select Name,Enabled | Export-Csv .\users.csv -NoTypeInformation
