Mini laburi ghidate

Lab 1: Active Directory rapid

  1. Instaleaza AD DS pe Windows Server 2022.
  2. Promoveaza serverul ca Domain Controller nou.
  3. Creeaza OU, utilizator si grup de test.
Install-WindowsFeature AD-Domain-Services -IncludeManagementTools
Install-ADDSForest -DomainName "lab.local"
New-ADOrganizationalUnit -Name "LabUsers" -Path "DC=lab,DC=local"

Lab 2: Hardening Windows 11

  1. Activeaza BitLocker pe drive-ul OS.
  2. Configureaza Windows Defender Firewall profile.
  3. Verifica statusul de update.
Enable-BitLocker -MountPoint "C:" -EncryptionMethod XtsAes256 -UsedSpaceOnly
Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled True
Get-HotFix | Sort InstalledOn -Descending

Lab 3: Onboarding Microsoft 365

  1. Conecteaza-te la Microsoft Graph PowerShell.
  2. Creeaza un user cloud de test.
  3. Aloca o licenta disponibila.
Connect-MgGraph -Scopes User.ReadWrite.All, Directory.ReadWrite.All
New-MgUser -DisplayName "Lab User" -UserPrincipalName "lab.user@contoso.onmicrosoft.com" -MailNickname "labuser" -AccountEnabled
Get-MgSubscribedSku

Lab 4: Azure VM + Monitoring

  1. Creeaza resource group pentru laborator.
  2. Provisioneaza o VM Windows.
  3. Verifica metricele de baza.
New-AzResourceGroup -Name rg-winlab -Location westeurope
New-AzVM -ResourceGroupName rg-winlab -Name vm-win11lab -Location westeurope
Get-AzMetricDefinition -ResourceId (Get-AzVM -Name vm-win11lab -ResourceGroupName rg-winlab).Id
