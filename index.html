Windows Server 2022 (Exact pe cele 9 PDF-uri)

00 - Inleiding / Introducere

Ce inseamna: diferenta dintre Windows Client si Windows Server, editii, tipuri de interfata, cerinte minime, schema laborator.

Meaning: differences between Windows Client and Windows Server, editions, interface types, minimum requirements, lab topology.

Pasi: valideaza hardware, alege editia, documenteaza topologia de laborator.

Steps: validate hardware, select edition, document lab topology.

Get-ComputerInfo | Select CsName,WindowsProductName,OsHardwareAbstractionLayer
systeminfo
Get-NetAdapter

Verificare: ai minim 1 VM Server cu resurse conforme si plan de retea clar.

Validation: you have at least one compliant Server VM and a clear network plan.

01 - OpzettenDomein / Domeniu AD DS

Ce inseamna: AD DS, forests, domains, OUs, FSMO, autentificare Kerberos, functional levels.

Meaning: AD DS, forests, domains, OUs, FSMO roles, Kerberos authentication, functional levels.

Pasi: instaleaza rolurile AD DS/DNS, promoveaza DC, valideaza replicare si DNS intern.

Steps: install AD DS/DNS roles, promote DC, validate replication and internal DNS.

Install-WindowsFeature AD-Domain-Services,DNS -IncludeManagementTools
Install-ADDSForest -DomainName "academy.local" -DomainNetbiosName "ACADEMY"
Get-ADForest
Get-ADDomain
dcdiag /v

Verificare: login in domeniu functioneaza, rezolvarea DNS interna raspunde corect.

Validation: domain sign-in works and internal DNS resolution is correct.

02 - Accounts / Obiecte in AD

Ce inseamna: security principals, OU planning, ADUC, RSAT, users/groups/computers, delegare.

Meaning: security principals, OU planning, ADUC, RSAT, users/groups/computers, delegation.

Pasi: proiecteaza OUs, creeaza utilizatori/grupuri, deleaga control pe OU unde este necesar.

Steps: design OUs, create users/groups, delegate OU control where needed.

New-ADOrganizationalUnit -Name "Workstations" -Path "DC=academy,DC=local"
New-ADUser -Name "Ana Pop" -SamAccountName apop -Enabled $true -AccountPassword (ConvertTo-SecureString "P@ssw0rd!" -AsPlainText -Force)
New-ADGroup -Name "GG-Helpdesk" -GroupScope Global -GroupCategory Security
Get-ADUser -Filter * -SearchBase "OU=Workstations,DC=academy,DC=local"

Verificare: obiectele apar in OU-urile corecte si drepturile delegate functioneaza.

Validation: objects are in the right OUs and delegated rights work as expected.

03 - File Server

Ce inseamna: NTFS permissions, ownership, encryption, quotas, file sharing, role File Server.

Meaning: NTFS permissions, ownership, encryption, quotas, file sharing, File Server role.

Pasi: creeaza structura share, aplica model AGDLP, configureaza quota/FSRM.

Steps: build share structure, apply AGDLP model, configure quota/FSRM.

Install-WindowsFeature FS-FileServer,FS-Resource-Manager -IncludeManagementTools
New-Item -Path C:\Shares\Finance -ItemType Directory
New-SmbShare -Name Finance$ -Path C:\Shares\Finance -FullAccess "Domain Admins"
icacls C:\Shares\Finance /grant "ACADEMY\GG-Finance-RW:(OI)(CI)M"
Get-SmbShare

Verificare: utilizatorii vad doar ce trebuie si cota de spatiu este aplicata.

Validation: users only access allowed data and storage quota is enforced.

04 - Gebruikersomgeving / User Environment

Ce inseamna: home folders, roaming profiles, logon scripts, Group Policy theory and processing order.

Meaning: home folders, roaming profiles, logon scripts, Group Policy theory and processing order.

Pasi: configureaza profil/homedrive, creeaza GPO baseline, aplica filtering (security/WMI).

Steps: configure profile/homedrive, create baseline GPO, apply security/WMI filtering.

New-GPO -Name "Baseline-Users"
New-GPLink -Name "Baseline-Users" -Target "OU=Users,DC=academy,DC=local"
Set-ADUser apop -HomeDirectory "\\SRV-DC01\Users$\apop" -HomeDrive "H:"
gpupdate /force
gpresult /r

Verificare: politicile se aplica in ordinea corecta, utilizatorul are mediul standardizat.

Validation: policies apply in the right order and user environment is standardized.

05 - Printerbeheer / Print Management

Ce inseamna: print server centralizat, cozi, drivere, drepturi, deploy prin GPO.

Meaning: centralized print server, queues, drivers, permissions, GPO deployment.

Pasi: adauga imprimanta, configureaza ACL, publica/deploy catre grupurile tinta.

Steps: add printer, configure ACL, publish/deploy to target groups.

Install-WindowsFeature Print-Server -IncludeManagementTools
Get-Printer
Add-PrinterPort -Name "IP_192.168.10.50" -PrinterHostAddress "192.168.10.50"
Add-Printer -Name "HR-Printer" -DriverName "Microsoft IPP Class Driver" -PortName "IP_192.168.10.50"

Verificare: clientii tintiti primesc imprimanta si pot tipari conform drepturilor.

Validation: targeted clients receive the printer and print according to permissions.

06 - Serverbeheer / Server Management

Ce inseamna: administrare remote (RDP), MMC personalizat, monitorizare procese/services, Event Viewer.

Meaning: remote administration (RDP), custom MMC, process/service monitoring, Event Viewer.

Pasi: activeaza RDP securizat, creeaza console administrative, monitorizeaza loguri critice.

Steps: enable secure RDP, build admin consoles, monitor critical logs.

Set-ItemProperty -Path 'HKLM:\System\CurrentControlSet\Control\Terminal Server' -Name fDenyTSConnections -Value 0
Enable-NetFirewallRule -DisplayGroup "Remote Desktop"
Get-Service | Where-Object {$_.Status -eq 'Running'}
Get-WinEvent -LogName System -MaxEvents 20

Verificare: accesul remote functioneaza, logurile ofera evenimentele de operare relevante.

Validation: remote access works and logs expose relevant operational events.

07 - Disk Management

Ce inseamna: initialize disk, partitii, basic vs dynamic, volumes, RAID, DiskPart.

Meaning: disk initialization, partitions, basic vs dynamic disks, volumes, RAID, DiskPart.

Pasi: initializeaza discuri, creeaza volume, valideaza litere/mount points si starea storage.

Steps: initialize disks, create volumes, validate drive letters/mount points and storage health.

Get-Disk
Initialize-Disk -Number 1 -PartitionStyle GPT
New-Partition -DiskNumber 1 -UseMaximumSize -DriveLetter F
Format-Volume -DriveLetter F -FileSystem NTFS -NewFileSystemLabel "DATA"
Get-Volume

Verificare: volumele sunt online, formatate corect si vizibile in sistem.

Validation: volumes are online, correctly formatted, and visible in the system.

Virtualbox Windows Server Router Lab

Ce inseamna: laborator adaptat cu 2 NIC (internet + LAN intern), IP plan si RAS de baza.

Meaning: adapted lab with 2 NICs (internet + internal LAN), IP plan, and basic RAS setup.

Pasi: seteaza Bridged + Internal Network, atribuie IP-uri, testeaza rutarea in lab.

Steps: configure Bridged + Internal Network, assign IPs, test routing in lab.

Get-NetIPConfiguration
New-NetIPAddress -InterfaceAlias "Ethernet 2" -IPAddress 10.10.10.1 -PrefixLength 24
Install-WindowsFeature RemoteAccess,Routing -IncludeManagementTools
Test-NetConnection 8.8.8.8

Verificare: clientii din LAN intern au conectivitate conform designului de laborator.

Validation: internal LAN clients have connectivity according to lab design.
