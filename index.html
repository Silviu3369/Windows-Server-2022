AD DS, DNS, DHCP

Configurezi infrastructura enterprise de baza: domeniu, rezolvare nume, lease management.

Build the enterprise base infrastructure: domain, name resolution, and lease management.

Install-WindowsFeature AD-Domain-Services,DNS,DHCP -IncludeManagementTools
Install-ADDSForest -DomainName "academy.local" -DomainNetbiosName "ACADEMY"
Add-DhcpServerInDC -DnsName "SRV-DC01.academy.local" -IPAddress 192.168.10.10
Add-DhcpServerv4Scope -Name "LAN" -StartRange 192.168.10.100 -EndRange 192.168.10.220 -SubnetMask 255.255.255.0

Identity, Groups, Permissions (AGDLP)

New-ADOrganizationalUnit -Name "Users" -Path "DC=academy,DC=local"
New-ADUser -Name "Ana Pop" -GivenName Ana -Surname Pop -SamAccountName apop -Enabled $true -AccountPassword (ConvertTo-SecureString "P@ssw0rd!" -AsPlainText -Force)
New-ADGroup -Name "GG-Finance-RW" -GroupScope Global -Path "OU=Users,DC=academy,DC=local"
Add-ADGroupMember -Identity "GG-Finance-RW" -Members apop

File Server & Print Server

Install-WindowsFeature FS-FileServer,Print-Server,FS-Resource-Manager -IncludeManagementTools
New-Item -ItemType Directory -Path C:\Shares\Finance
New-SmbShare -Name Finance$ -Path C:\Shares\Finance -FullAccess "Domain Admins" -ChangeAccess "GG-Finance-RW"
icacls C:\Shares\Finance /inheritance:r
icacls C:\Shares\Finance /grant "ACADEMY\GG-Finance-RW:(OI)(CI)M"

GPO, Security, Audit

Set-NetFirewallProfile -Profile Domain,Private,Public -Enabled True
auditpol /set /category:"Logon/Logoff" /success:enable /failure:enable
